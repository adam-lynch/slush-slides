  1. gulp-slides

    A slidedeck generator built with Gulp

  2. Setup the codebase

    npm run setup

    This will install and configure everything.

  3. Create some slides

    They're stored in src/slides

    # Hello, world!

>Michael Scott:
    Wayne Gretzky:
    >You miss 100% of the shots you don't take.

    GitHub-flavoured Markdown is supported.

  4. Generate the slides

    npm run compile

  5. Profit

    Open output/index.html
